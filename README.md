# Bayesian Modeling and Data Analysis Portfolio

A curated portfolio of Bayesian statistical modeling projects developed from advanced coursework and transformed into professional data science case studies.

## Overview

This repository demonstrates practical Bayesian workflows for statistical inference, posterior estimation, predictive model evaluation, regression analysis, and hierarchical modeling.

The projects focus on uncertainty quantification, probabilistic reasoning, computational Bayesian methods, and reproducible statistical analysis.

---

## Projects

Notebook:

Topics:
- Importance Sampling
- Markov Chain Monte Carlo (MCMC)
- Hamiltonian Monte Carlo (HMC)
- Posterior diagnostics
- Prior sensitivity analysis
